Create Codes

Use this feature to create an assessment codes for your candidate. Each code can only be used once. You can create one or several codes at a time.

Search Codes

Search by Last Name or UserID to find a candidate record. Allows you to see if a candidate has completed the assessment.

View Codes

Use this feature to view assessment codes that you have created. View Used or Unused codes in a list.
